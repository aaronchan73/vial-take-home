# Vial Take Home Assignment
## Running the Application
### Frontend
- Run the following:
```sh
cd frontend && npm i && npm run dev
```
- Frontend is running on `http://localhost:5173/`

### Backend
- Run the following:
```sh
cd backend && npm i && npm run dev
```
- Backend is running on `http://localhost:3000/`
